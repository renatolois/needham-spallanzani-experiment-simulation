# Simulação de Needham e Spallanzani

> Esta simulação recria os experimentos clássicos de John Needham e Lazzaro Spallanzani, permitindo variar a temperatura e a obstrução do frasco para observar diferenças no surgimento de microrganismos. O objetivo é oferecer uma compreensão mais didática e interativa dos testes realizados por ambos para investigar a abiogênese, simulando aspectos como resistência e taxa de reprodução dos microrganismos por meio de valores e proporções fictícias.
